A GUI-based Banking Management System developed using Python Tkinter and MySQL for efficient management of customer banking records. The application provides functionalities such as account creation, customer data management, balance handling, and record updates through an interactive desktop interface. It implements CRUD operations with secure database connectivity and displays customer information dynamically using a tabular interface.

Features
Add new customer accounts
Update customer details
Delete customer records
Display customer information
Manage account type and balance
MySQL database integration
User-friendly GUI using Tkinter
Tech Stack
Python
Tkinter
MySQL

This project helped strengthen concepts in GUI development, database management, CRUD operations, and Python application design.
